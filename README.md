# vue-apiCloud-model-mobile

目前支持 vscode,其余未拓展。需安装 apiCloud-cli，vscode 的 apiCloud 插件配置同步路径
运行 npm run watch-build，每次编译打包生成 apiCloud 文件，利用 apiCloud-cli 执行 wifi 真机同步。
